Movie Api

Movie Api allows a web application to acess information about movies, directors, genres, and actors. Users will be able to sign up, update their information, and create a list of favourite movies.

Request URL HTTP Method Response
Get a list of movies /movies GET Returns a JSON list of movies
Get information about a specific movie /movies/:title GET Returns JSON data about the specific movie
Get information about a specific genre of a movie /genres/:title GET Returns json data of a description of the genre and a list of movies of that genre
Get information about a director /directors/:name GET Returns JSON data about a specific director
Let a new user register /user/:newuser POST Add new user data to the JSON data of users
Allow current users to update there account information /user/:username PUT Updates the user JSON data with the changes requested by the user
Allows user to add a movie to a list of their favourites /favourites POST Gives the user a message that the movies was added to favourites while updating the favourites list
Allows a user to remove a movie from their favourites /favourites/:title DELETE Gives the user a message that the movie has been removed while updating the favourites list
Allows a user to delete their account /user/:username DELETE Gives user a message that their account has been deleted while removing the account info from the users data
